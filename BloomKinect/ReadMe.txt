Toxic Bloom README

Created: 2015-7-29 by Chris Linder
Notes: This is a rough set of instructions that works
for me, but I might have many required SDKs and 
libraries already installed.  If this doesn't work you
please update this doc with the additional steps
needed.

Running The Program:
1) Install the Kinect Beta SDK KinectSDK64.msi
2) Run the program .\Release\BLOOM_XInputGame.exe

Building The Program:
1) Install the Kinect Beta SDK KinectSDK64.msi
2) Install Visual Studio C++ 2010 Express
3) Open the solution SkeletalViewer.sln
4) Build and run in either Debug or Release
